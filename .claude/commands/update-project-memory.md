请检查是否存在 PROJECT_CONTEXT.md 文件：

**如果不存在**：创建新的项目知识库文档，包含：
1. 快速概览（项目功能、技术栈）
2. 核心架构（2-3段清晰说明）
3. 重要目录和文件
4. 关键知识点（认证、数据库、配置等）
5. 常见开发任务和模式

**如果已存在**：根据本次对话智能更新：
- 添加新发现的重要知识
- 修正理解有误的内容
- 补充缺失的关键信息
- 删除过时或冗余的信息

要求：保持文档精简实用,不超过300行，只记录开发时真正需要的且最重要的内容。
